SISTEMA GERENCIAMENTO DE CONSULTAS

O projeto consiste em um sistema que será utilizado em clínicas (públicas ou particulares) de médicos gerais. Decidimos fazer a mudança de hospitais para clínicos gerais devido à grande complexidade dos hospitais e à ampla quantidade de documentações próprias nesses locais, o que dificultaria o desenvolvimento do sistema. Nosso objetivo é utilizar os conhecimentos que possuímos para desenvolver uma solução eficaz para uma necessidade específica.

Este sistema foi projetado para atender clínicas médicas, permitindo o gerenciamento eficiente de informações e facilitando a comunicação entre médicos e pacientes. A plataforma busca ser acessível, intuitiva e voltada para as necessidades reais dos usuários.
